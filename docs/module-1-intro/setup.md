# Module 1: Isaac Sim Setup

This module introduces you to NVIDIA Isaac Sim, a powerful robotics simulation environment. You'll learn how to set up your development environment and create your first simulated robot.

## Learning Objectives

- Install and configure Isaac Sim
- Understand the simulation environment
- Create basic robot models
- Run your first simulation

## Prerequisites

- NVIDIA GPU with CUDA support
- Isaac Sim installed
- Basic understanding of robotics concepts

## Topics Covered

- Isaac Sim installation and setup
- Simulation environment overview
- Basic robot models and assets
- Running your first simulation
