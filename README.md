# Project Structure

This project is organized into frontend and backend components:

## Frontend
- `frontend/docusaurus-project/` - Docusaurus documentation site for the educational modules

## Backend
- `backend/isaac_ros/` - Isaac ROS components
- `backend/isaac_ros_workspace/` - Isaac ROS workspace with VLA module
- `backend/isaac_sim/` - Isaac Sim components for humanoid robot simulation
- `backend/nav2/` - Navigation components for bipedal humanoid navigation

## Documentation
- `docs/` - General project documentation
- `specs/` - Project specifications
- `history/` - Project history and prompts
