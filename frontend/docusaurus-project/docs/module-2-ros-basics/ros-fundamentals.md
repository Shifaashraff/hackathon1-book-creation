# Module 2: ROS 2 Fundamentals

This module introduces you to Robot Operating System (ROS 2), the middleware framework for robotics applications. You'll learn how to create nodes, topics, services, and actions.

## Learning Objectives

- Understand ROS 2 architecture
- Create and run ROS 2 nodes
- Implement topics, services, and actions
- Debug ROS 2 applications

## Prerequisites

- Basic programming knowledge
- Module 1 completed
- ROS 2 Humble Hawksbill installed

## Topics Covered

- ROS 2 architecture and concepts
- Creating nodes and packages
- Topics and message passing
- Services and actions
- Debugging and tools
