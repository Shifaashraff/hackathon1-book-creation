# Module 2: Exercises

Complete these exercises to practice ROS 2 concepts learned in Module 2.

## Exercise 1: Basic ROS 2 Node

- Create a simple publisher node
- Create a subscriber node
- Exchange messages between nodes

## Exercise 2: Services

- Implement a custom service
- Create service client and server
- Test service communication

## Exercise 3: Actions

- Create a custom action
- Implement action client and server
- Test action-based communication

## Assessment

Complete all exercises and demonstrate proper ROS 2 communication patterns.
