# Module 2 Assessment

Assessment for Module 2: ROS 2 Fundamentals

## Knowledge Check

1. Explain the publisher-subscriber pattern in ROS 2.
2. What are the differences between services and actions?
3. How do you create a custom message type in ROS 2?

## Practical Exercise

Implement a ROS 2 system with the following components:
- A publisher node that sends robot sensor data
- A subscriber node that processes the data
- A service that provides robot status information
- Document your implementation and testing process

## Evaluation Criteria

- Proper node implementation
- Correct communication patterns
- Successful data exchange
- Clear documentation
