# Module 1 Assessment

Assessment for Module 1: Isaac Sim Introduction

## Knowledge Check

1. What is Isaac Sim and its primary purpose?
2. Name three key features of Isaac Sim.
3. How do you configure physics properties in Isaac Sim?

## Practical Exercise

Create a simple robot simulation in Isaac Sim with the following requirements:
- Load a robot model
- Configure basic properties
- Run a physics-based simulation
- Document your approach and results

## Evaluation Criteria

- Proper simulation setup
- Correct physics configuration
- Successful execution
- Clear documentation
