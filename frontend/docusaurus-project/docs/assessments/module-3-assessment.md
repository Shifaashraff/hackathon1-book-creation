# Module 3 Assessment

Assessment for Module 3: AI Perception and Navigation

## Knowledge Check

1. Explain the difference between SLAM and path planning.
2. What are the key components of a perception pipeline?
3. How does the navigation stack work in ROS 2?

## Practical Exercise

Create an AI perception and navigation system with the following components:
- Visual SLAM implementation
- Object detection pipeline
- Path planning algorithm
- Navigation controller
- Document your approach and results

## Evaluation Criteria

- Proper system architecture
- Successful perception pipeline
- Effective navigation
- Clear documentation
